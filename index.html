1.Press any key to start. Simon will give the first signal. Repeat the signal by pressing the same color box.
2.Simon will add one signal to previous sequence. Repeat all of the signals sequentially by pressing the same color boxes, in order.
3.Simon will add another signal to the existing sequence.
4.Continue playing as long as you can repeat the sequence of signals correctly. After the 5th, 9th and 13th signals in a sequence, Simon automatically speeds up.
5.If you enter any signal wrong, the game would be over and you'll need to start from the beginning.
Good Luck!
